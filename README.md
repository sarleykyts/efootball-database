# 🎮 eFoot Cards Database 2026

A premium, interactive web database inspired by the aesthetics of eFootball 2026. 

## Features
- ✨ Epic, Showtime, & Big Time animated holographic card borders
- 🌌 Particles.js interactive background + Parallax stadium
- 📊 Progress bar stat visualizations & Player Comparison mode
- 📱 Fully Responsive + PWA (Installable on mobile)
- 💾 LocalStorage Favorites saving
- 🌙 Dark/Light mode + Stadium BGM

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub named `efootball-database`.
2. Upload all these files directly to the `main` branch.
3. In your GitHub repository, go to **Settings** > **Pages**.
4. Under "Build and deployment", set the source to **Deploy from a branch**.
5. Select the `main` branch and click **Save**.
6. Wait 1-2 minutes, and your site will be live at `https://[your-username].github.io/efootball-database/`!
